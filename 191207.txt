npm : node packaging manager
js기반 코드를 재사용하고 공유하기 위해서 만들어져있다.

- 노드 서버 생성 -
터미널에서
빈 디렉토리를 만들고 mkdir 문서/파일명
$ npm init
name : nodeserver
description : '설명'

init 이 끝나면 packaging.json이 생성된다.
va ~파일명 하면 터미널에서 내용을 볼 수 있음.

node based webserver
express 설치 express --save
--save : 설치된  express를 pakcag.json에 넣어놓고 이 packaging를 받는사람이 이걸 설치할 수 있게 도와줌!

express 설치시 node_modules도 설치되는데, express 는 큰 fw이기 때문에 많은 모듈들이 설치됨

노드몬 설치
npm install nodemon --save
글로벌 설치
npm install nodemon -g
